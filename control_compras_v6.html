Configuración

Conecta la IA para el escaneo automático de facturas

Webhook URL de Make.com
Pega aquí la URL de tu webhook de Make.com. Esta URL es el puente entre la app y la IA de Claude para leer tus facturas automáticamente.
Guía de configuración en Make.com
1
Crea el webhook en Make.com
Ve a make.com → New Scenario → agrega módulo Webhooks → Custom webhook → clic en "Add" → copia la URL que te da.
2
Agrega módulo HTTP para llamar a Claude
Después del webhook → "+" → HTTP → Make a request. Configura así:
URL: https://api.anthropic.com/v1/messages Method: POST Headers: x-api-key: TU_API_KEY_DE_ANTHROPIC anthropic-version: 2023-06-01 content-type: application/json Body type: Raw Content type: application/json Body: (ver paso 3)
3
Body del request a Claude
Pega este JSON en el campo Body. Make reemplaza {{1.imageBase64}} con la imagen que envía la app:
{ "model": "claude-sonnet-4-20250514", "max_tokens": 800, "messages": [{ "role": "user", "content": [ { "type": "image", "source": { "type": "base64", "media_type": "{{1.mediaType}}", "data": "{{1.imageBase64}}" } }, { "type": "text", "text": "Analiza esta factura. Responde SOLO con JSON sin markdown:\n{\"proveedor\":\"nombre\",\"numero\":\"num factura\",\"fecha\":\"YYYY-MM-DD\",\"monto\":0.00,\"desc\":\"descripcion\",\"centro\":\"Cocina o Bebidas u Operaciones\",\"pago\":\"forma de pago\"}" } ] }] }
4
Configura la respuesta (Output)
Agrega un tercer módulo: Webhooks → Webhook response. En "Body" pon: {{2.data}} (la respuesta del HTTP). Activa el scenario y pega la URL del webhook arriba.
5
Obtén tu API Key de Anthropic
Ve a console.anthropic.com → API Keys → Create Key. Pégala en el header x-api-key del paso 2.
